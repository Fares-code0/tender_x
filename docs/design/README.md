# حزمة توثيق UX/Design — نظام المناقصات

> نظام إدارة المناقصات الذكي (Smart Tender Management System)

## الغرض

توثيق شامل ومرجعي لتصميم وسير عمل نظام المناقصات، يربط كل شاشة بدور وحالة وقاعدة عمل وإجراء عبر معرّفات تتبُّع مشتركة (traceability IDs).

- 🔷 **مخطّط**: قيد التخطيط/التطوير (M3+)

## المعرّفات المشتركة

### BR- (Business Rules)
قواعد العمل الأساسية التي تحكم سير العمل (BR-001، BR-002، وغيرها)

### ACT- (Actions)
الإجراءات المنفّذة من قبل المستخدمين في النظام (ACT-01 إلى ACT-13)

### SCR- (Screens)
الشاشات والواجهات في التطبيق (SCR-01 إلى SCR-06)

### JRN- (Journeys)
رحلات المستخدم من البداية إلى النهاية (JRN-01 إلى JRN-07)
